# Assignment operators

---
Assignment operators are used to assign values to variables:

| Operator | Name    | Example    |
|----------|---------|------------|
| =        | x = 6   | x = 6      |
| +=       | x+=3    | x = x + 3  |
| -=       | x-=3    | x = x - 3  |
| *=       | x*=3    | x = x * 3  |
| /=       | x/=3    | x = x / 3  |
| %=       | x%=3    | x = x % 3  |
| //=      | 	x//=3  | x = x // 3 |
| **=      | x**=3   | x = x ** 3 |
| &=       | x&=3    | x = x & 3  |
| ^=       | x^=3    | x = x ^ 3  |
| >>=      | x>>=3   | x = x >> 3 |
| <<=      | x<<=3   | x = x << 3 |

```python
x = 6

print(x + 3)
print( x - 3 )
print( x * 3 )
print( x / 3)
print( x % 3)
print( x // 3)
print( x ** 3)
print( x & 3 )
print( x ^ 3)
print( x >> 3)
print( x << 3)
```
